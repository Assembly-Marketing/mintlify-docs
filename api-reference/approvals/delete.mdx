---
title: 'Delete approval'
openapi: 'DELETE /approvals/{id}'
---

Delete an approval request. This removes the approval requirement from the post.

## Path parameters

<ParamField path="id" type="string" required>
  UUID of the approval to delete
</ParamField>

## Response

<ResponseField name="deletedApproval" type="object">
  The deleted approval object
</ResponseField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://app.tryordinal.com/api/v1/approvals/789e0123-e89b-12d3-a456-426614174004 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch(
  'https://app.tryordinal.com/api/v1/approvals/789e0123-e89b-12d3-a456-426614174004',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://app.tryordinal.com/api/v1/approvals/789e0123-e89b-12d3-a456-426614174004',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
```
